🛠️ Features

Multi-Provider AI Router

Automatic fallback chain: if one provider fails, it tries the next. No single point of failure.

Vibecode Actions

Natural-language driven code workflows: generate, explain, refactor, debug, convert, document, test, optimize

Autonomous Local Agent

Give the agents a goal and it plans and executes: scaffolding → model design → route generation → test writing → execution → self-correction

Model Context Protocol (MCP)

Exposes 15+ tools over stdio or HTTP for AI clients like Claude Desktop, Cursor, or any MCP-compatible agent

Session Condensation

Long conversations? Automatically condense 100+ message histories into a concise summary. Never lose context.

Security-First Design

Path traversal protection, config validation, API key encryption, and strict execution sandboxing

⚡ Install in 60 Seconds

No npm account needed. Clone and run:

# 1. Clone the repository
git clone https://github.com/black-tiger-computing/tiger-code-companion.git
cd tiger-code-companion

# 2. Install dependencies
npm install

# 3. Compile TypeScript
npm run compile

# 4. Run setup wizard (interactive model selection)
node src/cli.js setup

# 5. Start using it!
node src/cli.js chat

Quick Commands

tiger-code-pilot analyze src/app.js --mode security Analyze code for security issues
tiger-code-pilot vibecode generate "a REST API in Python" Generate code from natural language
tiger-code-pilot agent Start autonomous agent

🏗️ Architecture

The system consists of three main interfaces (CLI, VS Code Extension, Local Agent) that connect to a Core Engine (AI Router), which routes requests to multiple AI providers through an abstraction layer.
